# Mipe Crew

Landing Page institucional desenvolvida em React para uma empresa de produção audiovisual especializada no mercado automotivo.

## Tecnologias

- React
- Vite
- JavaScript
- CSS
- Responsividade
- Vercel

## Funcionalidades

- Hero responsivo
- Portfólio em vídeo
- CTA integrado ao WhatsApp
- Layout otimizado para desktop e mobile
